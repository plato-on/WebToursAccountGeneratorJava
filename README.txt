Генератор аккаунтов для наполнения базы данных WebTours

Открыть WebToursAccountGenerator.iml с помощью IntellIJIdea
Открыть артефакт WebToursAccountGenerator.jar в командной строке с помощью команды: java -jar "абсолютный\путь\к\файлу" 

Генерирует файл Accounts.dat и пользователей в собственной директории
Перенести данные нужно самостоятельно
Файл Accounts.dat - в корень каждого скрипта
Файлы username0.dat - в систему WebTours ..\WebTours\cgi-bin\users
Перезаписывает пользователей при повторном использовании
